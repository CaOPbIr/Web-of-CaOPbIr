### 步骤3：添加图片悬停放大效果
**位置**：`style.css`
**操作**：添加 `.paper-image` 悬停放大样式

**新增CSS代码**：
```css
.paper-image {
    width: 100%;
    height: auto;
    display: block;
    transition: transform 0.3s ease;
}

.paper-image:hover {
    transform: scale(1.05);
}
```

**注意**：需要检查现有 `.paper-image` 样式，如果已有 `transition` 属性，则只需添加 `:hover` 状态。
